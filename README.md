## 📌 Introduction

This is a responsive Weather Web Application built using HTML, CSS, and JavaScript. It fetches real-time weather data using the Visual Crossing Weather API and detects the user’s current location using a geolocation service. The app supports searching cities, hourly/weekly forecasts, temperature unit switching (Celsius/Fahrenheit), and dynamic UI updates based on weather conditions.

## ✨ Features
- 🌍 Auto-detect user location
- 🔎 Search any city with autocomplete suggestions
- 🌡️ Switch between Celsius and Fahrenheit
- ⏱️ Hourly and weekly forecast views
- 🌤️ Dynamic weather icons based on conditions
- 🌄 Background changes based on weather
- 💧 Humidity, UV index, wind speed, visibility, air quality display
- 📅 Live date and time update
- ⚡ Loader animation while fetching data
- 📱 Fully responsive design

## 🛠️ Technologies Used
- HTML5
- CSS3 (Flexbox, Responsive Design, Animations)
- JavaScript (ES6)
- Fetch API
- Visual Crossing Weather API
- Geolocation DB API

## 📦 Installation

1. Clone this repository:
```bash
git clone https://github.com/Rusira-Indrachapa/weather-app.git
```

2. Navigate into the project folder:
```bash
cd weather-app
```

3. Open index.html in your browser.

## 🚀 Usage

1. App auto-detects your city on load.
2. Search for any city using the search bar.
3. Click suggestions to autofill city names.
4. Toggle between Celsius/Fahrenheit.
5. Switch between hourly/weekly forecast.
6. View detailed weather info.

## 🌐 API Integration

### Visual Crossing Weather API
https://weather.visualcrossing.com/VisualCrossingWebServices/rest/services/timeline/

Add your API key in script.js:
const apiKey = "YOUR_API_KEY";

### Geolocation API
https://geolocation-db.com/json/
